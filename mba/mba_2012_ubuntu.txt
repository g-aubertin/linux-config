### modules to install

macfanctld : fan control daemon for intel based macbook
 sudo add-apt-repository ppa:mactel-support/ppa
 sudo apt-get install macfanctld
 sudo apt-get install applesmc-dkms (updated applesmc module for 2012 mba)
 sudo modprobe applesmc

lightum : light sensor daemon
 sudo add-apt-repository ppa:poliva/lightum-mba
 sudo apt-get install lightum lightum-indicator

more general packages :
 sudo apt-get install lm-sensors

### post-install configuration

macfanctld: you need to check which sensors are bogus with 'sensors', look at
its associated number in macfanctld (using 'macfanctld -f') and add it to the
exclude list (/etc/macfanctld.conf)
